# World Conqueror Strategy

## Overview
A turn-based strategy game inspired by World Conqueror 4, built entirely with GDevelop's visual event system. No traditional coding required.

## How to Play
1. Import `WorldConquerorStrategy.json` into [GDevelop](https://editor.gdevelop.io)
2. Click Preview
3. Tap your units (blue) to select, tap a region to move, tap enemies (red) to attack

## Tech Stack
- **Engine**: GDevelop 5 (Visual Programming)
- **Platforms**: Android, iOS, Web (HTML5)
- **Style**: 2.5D top-down strategy

## Features
- 3 unit types with unique stats
- 6 terrain types with strategic modifiers
- AI opponent
- Economy system with city income
- Mobile-first touch controls
