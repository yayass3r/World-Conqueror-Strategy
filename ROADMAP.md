# World Conqueror Strategy - Roadmap

- [x] Turn-based movement system
- [x] Combat with damage calculations
- [x] AI opponent (basic)
- [x] Economy (gold + city income)

- [x] Mobile touch controls
- [x] UI (info panel, gold display, end turn)
