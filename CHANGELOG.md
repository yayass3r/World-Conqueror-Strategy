# World Conqueror Strategy - Changelog

## [1.0.0] - 2026-04-03
### Added
- Core turn-based strategy gameplay
- 3 unit types: Infantry, Tank, Aircraft
- 6 terrain types: Plains, City, Desert, Mountain, Forest, Sea
- 20-region strategic map (5x4 grid)
- AI opponent with basic strategy
- Economy system (gold generation from cities)
- Combat system with damage formulas and terrain bonuses
- Mobile touch controls
- Unit info panel
- Health bars above units
- Selection markers and visual feedback
- Damage text popups
- Explosion effects for unit destruction
- Turn counter and phase display
- End Turn button
- Win/loss conditions
- Complete 23-page development guide
- Game balance spreadsheet with all formulas

### Technical
- GDevelop 5 project (version 4.0.145)
- 22 game objects
- 13 event groups
- 2 layers (Base + UI)
- Scene variables state machine
- Timer-based AI sequencing
- Instance-based object variables
